# Keyboard Shortcuts

## Global

| Key | Action |
|---|---|
| `1`-`4` | Select menu item directly |
| `↑`/`↓` or `j`/`k` | Navigate up/down |
| `Enter` / `Space` | Select / Confirm |
| `Tab` | Next field |
| `Shift+Tab` | Previous field |
| `Esc` | Back to menu |

## Screens

| Key | Action |
|---|---|
| `S` | Open Settings |
| `W` | Open Wallet Manager |

## Settings

| Key | Action |
|---|---|
| `←`/`→` or `h`/`l` | Change value (chain selector, key mode toggle) |
| `T` | Toggle private key visibility |
| `Ctrl+S` | Save config to disk |
| `Esc` | Back to menu |

## Disperse Form

| Key | Action |
|---|---|
| `↑`/`↓` | Switch between chain / wallet list |
| `Enter` | Edit focused text field |
| `Esc` | Cancel edit / Back |

## Fee Calculator

| Key | Action |
|---|---|
| `↑`/`↓` | Switch chain |
| `Enter` | Edit recipients / amount field |
| `Tab` | Next field |

## Wallet Manager

| Key | Action |
|---|---|
| `G` | Generate wallets |
| `S` | Save wallet list to disk |
| `Enter` | Edit focused field |
| `Tab` | Switch between fields |
